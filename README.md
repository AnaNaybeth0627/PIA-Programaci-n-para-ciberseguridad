Este proyecto se centra en el monitoreo de seguridad de los archivos del sistema operativo, ofreciendo múltiples tareas que ayudan a la detección de actividad sospechosa, modificaciones en archivos críticos y procesos inusuales que puedan ocurrir en el sistema. 

La Tarea 1 implementa un script modular en PowerShell que monitorea en tiempo real los cambios en los archivos del usuario dentro de “C:\Users\usuario”, registrando eventos de creación, modificación, eliminación y renombrado. Los resultados se muestran en la terminal y se guardan tanto en un archivo de texto legible (reporte_de_cambios.txt) como en un log estructurado en formato JSON lines (reporte_de_cambios.json), incluyendo información de timestamp, run_id, módulo, nivel, evento y detalles. Este componente cumple con los requisitos de logging, evidencia reproducible y ejecución controlada, integrándose fácilmente al menú principal del proyecto.
